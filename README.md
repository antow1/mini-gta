# Mini Vice Mobile - iPhone

Jeu HTML/CSS/JavaScript adapté au tactile.

## Comment jouer sur iPhone

Le plus simple :
1. Mets ce dossier en ligne avec GitHub Pages, Netlify ou Vercel.
2. Ouvre le lien dans Safari.
3. Safari → Partager → Ajouter à l’écran d’accueil.

## Contrôles

- Joystick gauche : marcher / conduire
- ACTION : interaction proche
- VOITURE : entrer / sortir
- MISSION : lancer une mission près d’un marqueur jaune
- RESET : recommencer

## Contenu

- Ville ouverte 2D
- Voitures
- Piétons
- Police

- Argent
